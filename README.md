# Auto Swipe Luxury

Premium mobile UI for choosing cars with swipe mechanics, built with React Native and Expo.

`Auto Swipe Luxury` is a concept app for a high-end automotive marketplace: users browse premium cars, swipe through offers, save favorites to a garage, and open detailed specs in a polished iOS-style interface.

## Highlights

- Tinder-like swipe flow for discovering cars
- Dark luxury visual style with glassmorphism cards
- Large editorial car photography
- Smooth card animations and expressive micro-interactions
- Russian UI copy and mobile-first layout
- Garage/favorites screen for saved cars
- Profile and filter screens
- Modern typography with Manrope and Space Grotesk

## Screens

- Onboarding with a premium hero image
- Discover deck with animated car cards
- Swipe reject/like states
- Match-style save confirmation
- Garage with saved cars
- Car detail screen with technical specifications
- Profile and filters

## Tech Stack

- React Native
- Expo
- React Native Web
- Expo Vector Icons
- Expo Linear Gradient
- Expo Blur
- Expo Google Fonts

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run web
```

Or start Expo normally:

```bash
npm start
```

## Version

Current release: `2.0.0`

## Concept

The app is designed as a premium automotive product rather than a generic marketplace. The interface focuses on fast decisions, strong visual hierarchy, dark surfaces, soft shadows, rounded cards, and a distinctive accent system to make each car feel like a curated offer.
